Jemuran Otomatis

Alat dan Bahan Jemuran Otomatis

  • Arduino Uno: Arduino adalah mikrokontroler yang berfungsi sebagai "otak" dari sistem jemuran otomatis. Alat ini akan menerima data dari sensor hujan dan remote IR, kemudian menggerakkan motor servo untuk membuka atau menutup jemuran sesuai kondisi cuaca.
  • Sensor Hujan: Sensor ini mendeteksi kehadiran air hujan. Saat mendeteksi air, sensor akan mengirimkan sinyal ke Arduino untuk mengaktifkan proses penutupan jemuran secara otomatis.
  • Servo Motor: Servo digunakan untuk menggerakkan atap jemuran otomatis. Motor ini mengatur posisi jemuran (membuka atau menutup) sesuai instruksi dari Arduino.
  • Remote Infrared (IR) dan IR Receiver: Remote digunakan untuk mengontrol jemuran secara manual, misalnya untuk membuka atau menutup jemuran atau mengatur kondisi lampu taman.
  • LED dan Resistor: LED sebagai indikator status pada sistem jemuran, seperti warna merah untuk menandakan kondisi hujan dan hijau untuk cuaca cerah.
  • LCD (Liquid Crystal Display): LCD menampilkan status sistem jemuran seperti "Hujan datang," "Tutup atap bro," dan informasi lain terkait kondisi jemuran.
  • Baterai atau Sumber Daya Listrik: Digunakan untuk memasok daya pada Arduino dan komponen lainnya.

Pengertian dan Kegunaan Jemuran Otomatis

Jemuran otomatis adalah sistem elektronik yang memanfaatkan sensor, motor, dan mikrokontroler untuk melindungi jemuran dari hujan secara otomatis. Tujuan utama dari sistem ini adalah menjaga pakaian agar tetap kering saat hujan dan tetap membuka jemuran ketika cuaca cerah. Sistem ini dirancang agar dapat bekerja secara otomatis dengan bantuan sensor atau melalui kontrol manual menggunakan remote.

Cara Kerja Jemuran Otomatis

  1. Deteksi Cuaca: Ketika sensor hujan mendeteksi air, sensor akan mengirim sinyal ke Arduino. Arduino kemudian mengaktifkan LED merah sebagai tanda bahwa hujan telah terdeteksi.
  2. Gerakan Jemuran: Arduino akan mengirim perintah ke motor servo untuk menggerakkan atap jemuran ke posisi menutup, sehingga jemuran terlindungi dari hujan.
  3. Kontrol Manual: Pengguna dapat menggunakan remote IR untuk membuka atau menutup jemuran secara manual jika diperlukan. Remote juga dapat mengontrol pencahayaan di sekitar jemuran, seperti mengaktifkan atau mematikan lampu taman.
  4. Tampilan Status: LCD akan menampilkan informasi tentang status jemuran, seperti "Hujan datang," "Tutup atap," atau "Tidak ada perintah" ketika tidak ada aksi yang dideteksi.

Contoh Program Servo Arduino

Contoh 1: Menggerakkan Servo pada Sudut Tertentu


#include  // Memanggil library servo

Servo myServo; // Mendeklarasikan servo dengan nama myServo

void setup() {
  myServo.attach(9); // Menghubungkan servo ke pin digital 9
}

void loop() {
  myServo.write(0);   // Menggerakkan servo ke 0 derajat
  delay(1000);        // Menunggu 1 detik
  myServo.write(90);  // Menggerakkan servo ke 90 derajat
  delay(1000);        // Menunggu 1 detik
  myServo.write(180); // Menggerakkan servo ke 180 derajat
  delay(1000);        // Menunggu 1 detik
}

Contoh 2: Menggerakkan Servo dengan Potensiometer


#include 

Servo myServo;  // Membuat objek servo
int potPin = A0;  // Pin potensiometer terhubung ke A0

void setup() {
  myServo.attach(9);  // Menghubungkan servo ke pin 9
}

void loop() {
  int potValue = analogRead(potPin);        // Membaca nilai potensiometer (0-1023)
  int angle = map(potValue, 0, 1023, 0, 180); // Mengonversi nilai ke sudut (0-180)
  myServo.write(angle);                     // Mengatur sudut servo sesuai nilai potensiometer
  delay(15);                                // Menunggu sedikit untuk pergerakan servo
}

Tips Menggunakan Servo dengan Arduino

  • Pastikan catu daya (VCC) sesuai spesifikasi servo. Umumnya servo membutuhkan daya 5V-6V.
  • Jangan terlalu sering menggerakkan servo pada sudut-sudut ekstrem (0° atau 180°) karena dapat menyebabkan kerusakan jika servo berusaha untuk melampaui sudut tersebut.
  • Jika proyek Anda menggunakan banyak servo, gunakan sumber daya eksternal agar tidak membebani Arduino.

Aplikasi Servo Motor

  • Lengan robot: Untuk menggerakkan sendi-sendi robot secara presisi.
  • Kamera Pan-Tilt: Mengatur arah kamera untuk menangkap sudut pandang berbeda.
  • Pengendali Pintu: Membuka dan menutup pintu otomatis, seperti pada mesin antrean atau alat pertanian.

Kesimpulan: Menggunakan servo dengan Arduino cukup mudah dan bisa dilakukan dengan beberapa baris kode sederhana. Dengan servo, kita dapat membuat berbagai proyek robotik dan mekanis yang menarik dan bermanfaat. Selalu perhatikan catu daya dan torsi yang dibutuhkan, terutama saat menggunakan servo pada aplikasi berat.

1. Inisialisasi dan Deklarasi Variabel

#include <IRremote.h>
#include <Servo.h>
#include <Wire.h>
#include <LiquidCrystal_I2C.h>
LiquidCrystal_I2C lcd(0x27, 16, 2); - Membuat objek LCD pada alamat I2C 0x27, ukuran 16x2.

  • IRremote: Library untuk mengontrol penerima IR agar dapat menerima sinyal dari remote.
  • Servo: Library untuk mengendalikan motor servo.
  • Wire dan LiquidCrystal_I2C: Untuk mengontrol LCD dengan komunikasi I2C.

2. Definisi Pin

Kode ini mendefinisikan pin-pin untuk komponen:

      const int LED_HIJAU = 4;
      const int LED_MERAH = 3;
      const int LED_HIJAU1 = 2;
      const int LED_HIJAU2 = 5;
      const int LED_HIJAU3 = 6;
      const int LED_HIJAU4 = 7;
      const int IR_PIN = 12;
      int rainsensor = A0;
      int servoPin = 9;
      int targetPos = 80;
  • LED_HIJAU dan LED_MERAH: Untuk lampu indikator cuaca.
  • LED_HIJAU1 hingga LED_HIJAU4: Untuk mengontrol lampu taman.
  • IR_PIN: Untuk menerima sinyal IR dari remote.
  • rainsensor: Sensor hujan pada pin analog A0.
  • servoPin: Untuk mengendalikan motor servo pada pin 9.

3. Variabel Servo, IR Receiver, dan Nilai Sensor Hujan

      Servo myServo;
      IRrecv irrecv(IR_PIN);
      decode_results hasil;
      int nilairain = 0;
      int jumlah_air = 700;
      int pos = 0;
      int stepDelay = 20;
  • myServo: Objek untuk mengontrol gerakan servo.
  • irrecv: Untuk menerima sinyal IR dari remote.
  • jumlah_air: Nilai ambang batas deteksi hujan.

4. Fungsi setup()

      void setup() { 
        Serial.begin(9600);
        irrecv.enableIRIn();
        myServo.attach(servoPin);
        lcd.init();
        lcd.backlight();
        pinMode(LED_HIJAU, OUTPUT);
        pinMode(LED_MERAH, OUTPUT);
        pinMode(LED_HIJAU1, OUTPUT);
        pinMode(LED_HIJAU2, OUTPUT);
        pinMode(LED_HIJAU3, OUTPUT);
        pinMode(LED_HIJAU4, OUTPUT);
      }
  • Serial.begin(9600): Memulai komunikasi serial untuk debugging.
  • irrecv.enableIRIn(): Mengaktifkan penerima IR.
  • myServo.attach: Menghubungkan servo pada pin yang ditentukan.
  • lcd.init() dan lcd.backlight(): Menginisialisasi dan mengaktifkan backlight LCD.
  • pinMode: Mengatur pin-pin untuk LED sebagai output.

5. Fungsi loop()

Bagian utama yang berjalan berulang-ulang:

a. Membaca Sensor Hujan
      nilairain = analogRead(rainsensor);
      Serial.print("Nilai sensor: ");
      Serial.println(nilairain);

Membaca nilai dari sensor hujan, dan mendeteksi hujan jika nilainya lebih rendah dari jumlah_air (700).

b. Deteksi Hujan dan Pengaturan Servo
      if (nilairain < jumlah_air) { 
        digitalWrite(LED_HIJAU, LOW);
        digitalWrite(LED_MERAH, HIGH);
        lcd.setCursor(0, 1);
        lcd.print("Hujan datang bro");
        delay(2000);
        myServo.write(80);
        delay(20);
      }

Jika hujan terdeteksi, LED merah menyala dan pesan ditampilkan di LCD. Servo akan bergerak ke posisi 80 untuk menutup atap.

c. Mengatur Remote IR
      if (irrecv.decode(&hasil)) {
        if(hasil.value == 16724175) {
          // Membuka atap, nyalakan LED hijau
        } else if(hasil.value == 16718055) {
          // Menutup atap, nyalakan LED merah
        } else if(hasil.value == 16743045) {
          // Nyalakan lampu taman
        } else if(hasil.value == 16716015) {
          // Matikan lampu taman
        }
        irrecv.resume();
      }
  • irrecv.decode: Mendeteksi sinyal dari remote control.
  • hasil.value: Menyimpan kode sinyal. Beberapa kode:
    • 16724175: Membuka atap dan menyalakan LED hijau.
    • 16718055: Menutup atap dan menyalakan LED merah.
    • 16743045: Menyalakan lampu taman.
    • 16716015: Mematikan lampu taman.
6. Menampilkan Pesan di LCD
      lcd.setCursor(0, 1);
      lcd.print("Hujan datang bro");

Menampilkan pesan agar pengguna mengetahui bahwa hujan telah terdeteksi.

Kesimpulan

Program ini mengontrol kondisi atap dan lampu secara otomatis atau manual dengan fitur utama:

  • Sensor Hujan: Menutup atap secara otomatis jika hujan.
  • Servo Motor: Mengontrol pergerakan atap.
  • Remote IR: Kontrol manual untuk membuka/menutup atap dan mengatur lampu taman.
  • LCD: Menampilkan status sistem.
