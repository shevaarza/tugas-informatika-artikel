Simulasi Traffic Light dengan Arduino

Alat dan Bahan

  • Arduino Uno (atau board Arduino lain)
  • LED (3 buah: merah, kuning, hijau)
  • Resistor 330Ω atau 220Ω (3 buah)
  • Breadboard
  • Jumper wires

Skema Rangkaian

Berikut adalah cara menyusun komponen untuk proyek Traffic Light dengan Arduino:

  • Sambungkan LED merah ke pin D13 pada Arduino melalui resistor 330Ω, dan hubungkan kaki katoda ke GND.
  • Sambungkan LED kuning ke pin D12 pada Arduino melalui resistor 330Ω, dan hubungkan kaki katoda ke GND.
  • Sambungkan LED hijau ke pin D11 pada Arduino melalui resistor 330Ω, dan hubungkan kaki katoda ke GND.

Program Arduino

Program berikut akan mengatur urutan nyala dan mati LED untuk simulasi lampu lalu lintas. Setiap lampu memiliki durasi tertentu untuk menyala seperti lampu lalu lintas asli.


        int redLED = 13;
        int yellowLED = 12;
        int greenLED = 11;
        
        void setup() {
          pinMode(redLED, OUTPUT);
          pinMode(yellowLED, OUTPUT);
          pinMode(greenLED, OUTPUT);
        }
        
        void loop() {
          // Nyala merah selama 5 detik
          digitalWrite(redLED, HIGH);
          digitalWrite(yellowLED, LOW);
          digitalWrite(greenLED, LOW);
          delay(5000);  // 5 detik
        
          // Nyala kuning selama 2 detik
          digitalWrite(redLED, LOW);
          digitalWrite(yellowLED, HIGH);
          digitalWrite(greenLED, LOW);
          delay(2000);  // 2 detik
        
          // Nyala hijau selama 5 detik
          digitalWrite(redLED, LOW);
          digitalWrite(yellowLED, LOW);
          digitalWrite(greenLED, HIGH);
          delay(5000);  // 5 detik
        
          // Ulangi kembali ke awal
        }

Penjelasan Program

  • setup(): Mengatur setiap pin LED sebagai output agar dapat dikendalikan oleh Arduino.
  • loop(): Mengatur urutan nyala dan mati LED sesuai durasi yang telah ditentukan (merah 5 detik, kuning 2 detik, hijau 5 detik).
  • delay(): Menentukan waktu tunggu agar LED tetap menyala pada durasi yang diinginkan.

Uji Coba dan Troubleshooting

  • Pastikan semua kabel dan resistor terhubung dengan benar.
  • Ubah nilai delay() jika ingin menyesuaikan durasi nyala lampu.

Pengembangan Lebih Lanjut

  • Tambahkan sensor inframerah untuk mendeteksi kendaraan dan menyesuaikan waktu lampu hijau secara otomatis.
  • Tambahkan tombol pedestrian untuk lampu pejalan kaki.

Kesimpulan

Membuat simulasi lampu lalu lintas dengan Arduino cukup mudah dan dapat dilakukan dengan beberapa baris kode. Proyek ini dapat digunakan sebagai dasar untuk aplikasi sistem kontrol lampu lalu lintas yang lebih kompleks.
